# 🎮 Fun App – You Have to Say YES! 😂

A fun little interactive web app where saying **"No"** isn't as easy as it sounds! 😆

Every time you try to click the **No** button, it jumps to a new random position while the chibi character gets angrier. If you finally click **Yes**, you'll unlock a fun celebration with music, animations, and an optional QR code section.

## 🌐 Live Demo

🔗 https://spectacular-licorice-b371bb.netlify.app/


---

## ✨ Features

- 🎯 Moving **No** button that avoids every click.
- 😡 Chibi character changes emotions with every failed attempt.
- 🎉 Celebration popup after clicking **Yes**.
- 🎵 Celebration music.
- 📱 Optional QR code & payment section.
- 📱 Responsive design for desktop and mobile.
